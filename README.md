
# Online Food Ordering Website

A full-stack web application built using the MERN (MongoDB, Express.js, React.js, Node.js) stack that allows users to browse restaurant menus, add food items to their cart, and place real-time orders.
The platform features a user-friendly frontend, secure backend APIs, and an admin dashboard for menu and order management.
